Projet : Chatbot avec python  
Nom-Prenom : Benaissa Sidahmed

Description du projet :
    C'est un Chatbot qui permet de répondre à des questions concernant les régles du jeu de football. 

Réalisations : 
    1 - Un réseau de neuronnes de type (FFNN) qui prend en entrée les bags de mots pour chaque pattern et qui retourne la probabilité 
        pour chaque tag.
    2 - Le type d'apprentissage : non supervisé.
    3 - Utilisation d'un fichier .json pour stocker les patterns, les tags et les réponses.
    4 - Interface graphique avec la bibliothéque tkinter.     

Améliorations possibles :
    1 - Utilisation d'un fichier .json contenant plus de données.
    2 - Utilisation d'un apprentissage plus amélioré.

Bibliothéque :
    - pip install nltk
    - pip install torch 
    - sudo apt-get install python3-tk
